### Notes

Stacks and queus help solve problems by putting a set of restrictions on the usage of arrays (or similar constructs).

They help make code clean and readable by following tried and true patterns that everyone understands.

Stacks: Last In First Out
- add to top
- look at top
- remove from top

Queues: First In First Out
- add to back
- remove from front
